Dans le cadre de notre 4ème année, la filière TIC de l'EPF nous a invité à réaliser un site internet permettant l'affichage de données de notre choix. Nos préférences nous ont rapidement menées vers l'étude du système solaire et des planètes qui le composent.

Les donées présentées concernent l'ensemble des paramètres connus et présentant un quelquonque intérêt

Pour parvenir à un affichage plaisant et clair, nous avons utilisé plusieurs outils de dévellopement web permettant un traitement. Ces derniers sont des frameworks ayant plusieurs utilités :

Boostrap permet de gèrer le design du site afin de présenter les infos le plus esthéquiquement possible. Il permet également de gerer la partie responsive du site permettant d'y accéder via des appareils style smartphones ou tablettes.

Angular permet de gérer l'interactivités du site via diverses directives ( comme par exemple : ng-show, ng-repeat, ng- click, etc... )

Utilisation: Les boutons des planètes permettent de visualiser toutes les données de chaque planète et la partie avec les check box permettent de visualiser les mêmes données de chaque planète afin de pouvoir les comparer.
